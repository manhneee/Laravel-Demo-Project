# Installation

Clone the repository
```
git clone <repository-url>
cd <project-folder>
```
Install dependencies

```
composer install
```
Copy environment file

```
cp .env.example .env
```
Generate application key

```
php artisan key:generate
```
Start the development server:

```
php artisan serve
```
-------------------------------
Updating Dependencies (Optional)
```
composer update
```
